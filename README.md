# Weather App
<img src="https://1.bp.blogspot.com/-838rnhFmnvU/WQvvHcXaIBI/AAAAAAABEEw/guhTlPRdqe0kHuN99Tgg22KCKUqkFhjvwCLcB/s400/job_itamae_woman.png" height="400px">
This is weather app which is made on Ruby on Rails, Bootstrap Navbar and Partials. This app gives the weather condition of Tokyo and Kathmandu.
This App gives an idea of:
- Weather API Introhttps
- Connect To API
- API Error Handling
- Bootstrap Navbar 

Weather Api (data) of Tokyo is taken from
URL: 
http://api.openweathermap.org/data/2.5/weather?q=tokyo&appid=6714466a75ef7842af9d373139741d1d

Weather Api (data) of Kathmandu is taken from
URL:
http://api.openweathermap.org/data/2.5/weather?q=kathmandu&appid=6714466a75ef7842af9d373139741d1d

Installation Procedure:
git clone
bundle install
rails s
